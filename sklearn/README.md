# Laboratory on Scikit-learn and Supervised Learning

This repository contains material and code for a laboratory session on Machine Learning with Python, focusing on **Scikit-learn** and **supervised learning**.  

## Overview

The notebook guides you through:
- Basic Python libraries for data science  
- Supervised learning algorithms:
  - Support Vector Machines (SVM)
  - Decision Trees
  - Random Forests
- Feature mapping and kernel methods
- Cross-validation and model evaluation
- Data preprocessing and visualization

---

## Prerequisites

### Requirements

Make sure you have the following installed:

- Python ≥ 3.7  
- [NumPy](https://numpy.org/install/)  
- [SciPy](https://scipy.org/install.html)  
- [Matplotlib](https://matplotlib.org/stable/users/installing.html)  
- [Pandas](https://pandas.pydata.org/docs/getting_started/index.html)  
- [Scikit-learn](https://scikit-learn.org/stable/install.html)  
- [Seaborn](https://seaborn.pydata.org/installing.html)

## Topics Covered

- Data manipulation using `NumPy` and `Pandas`
- Data visualization with `Matplotlib` and `Seaborn`
- Classification models:
  - SVM (linear, kernel, and feature mapping)
  - Decision Tree
  - Random Forest
- Model evaluation:
  - Accuracy, Precision, Recall, and F1 Score
  - Confusion Matrix
  - K-Fold Cross-Validation
- Hyperparameter optimization
